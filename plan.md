# 《三角洲行动》干员测试 - 计划与设计方案

## 1. 整体目标
将原有的“SBTI 人格测试”改造为《三角洲行动》专属的“干员测试”。玩家通过完成一系列基于战场剧情和战术决策的题目，最终测出自己最匹配的《三角洲行动》干员。

## 2. 主题与界面改造
- **标题**：“SBTI 人格测试” 更改为 “《三角洲行动》干员招募评估”
- **视觉风格**：
  - 将原本的小清新绿色调（`--accent: #6c8d71`）替换为具有战术感和科技感的“三角洲”主题色（例如深灰色、战术黑以及高亮橙色/亮蓝色）。
  - 背景、卡片增加一些科技感纹理或暗色主题设计。
- **文案风格**：从幽默嘲讽转变为硬核战术、军事风与适度的特种兵幽默。

## 3. 维度设计（保留15维度计算逻辑，但重命名）
原15维度将被赋予“战术评估”的新含义：
- **S (自我模型) -> 战术心理素质 (S)**: 
  - S1 战斗自信、S2 战术清晰度、S3 核心意志
- **E (情感模型) -> 团队信任度 (E)**: 
  - E1 后背交托、E2 团队羁绊、E3 独立与依赖
- **A (态度模型) -> 战场世界观 (A)**: 
  - A1 敌意感知、A2 战术灵活性、A3 战斗意义
- **Ac (行动驱力模型) -> 行动风格 (Ac)**: 
  - Ac1 进攻动机、Ac2 决策速度、Ac3 执行力度
- **So (社交模型) -> 协同与沟通 (So)**: 
  - So1 战术主动性、So2 阵型边界感、So3 虚实与欺骗

## 4. 角色（干员）映射设计
原有的27种人格将精简或映射为《三角洲行动》的经典干员及特殊彩蛋，补齐完整的 27 个人格：

**主要常规干员（14名）**：

1. **红狼 (凯·席尔瓦)**：高机动、高爆发、自信果断。（突击）

2. **牧羊人 (泰瑞·缪萨)**：稳重、重视防御与团队保护。（工程）

3. **露娜 (金卢娜)**：敏锐、擅长情报收集与分析。（侦察）

4. **蜂医 (罗伊·斯米)**：共情力强、团队支援核心。（支援）

5. **威龙 (王宇昊)**：执行力极强、敢打敢拼。（突击）

6. **骇爪 (麦晓雯)**：隐秘、战术灵活、善于欺骗敌人。（侦察）

7. **乌鲁鲁 (大卫·费莱尔)**：坚如磐石、火力压制、阵地核心。（工程）

8. **银翼 (兰登·穆勒)**：追踪索敌、战区情报收集。（侦察）

9. **蝶 (莉娜·范德梅尔)**：无人机治疗、远程补给。（支援）

10. **比特 (拉希德·拉哈尔)**：智能陷阱、哨兵母巢控场。（工程）

11. **佐娅 (蛊)**：激素增益、致盲毒雾、辅助控制。（支援）

12. **深蓝 (阿列克谢·彼得罗夫)**：防爆盾推进、钩爪索敌。（工程）

13. **无名 (Nameless)**：静默潜袭、闪光突进、反侦察。（突击）

14. **阿萨拉卫士 (Asara Guard)**：坚守阵地、不屈不挠的战士。（彩蛋性质的NPC阵营干员）

**特殊/隐藏角色及热梗（13名）**：

15. **曼德尔砖 (MandelBrick)**：极度贪财、只认高价值物资。（彩蛋触发）

16. **摸金校尉 (Runner)**：跑刀仔、垃圾佬、不爱打架只爱捡漏。（匹配兜底）

17. **非洲之星 (African Star)**：运气极差、次次开箱都是白板。

18. **海洋之泪 (Tear of Ocean)**：极度罕见的欧皇、随便摸一下就是天价物资。

19. **六套战神 (T6 Juggernaut)**：全身最顶级装备、莽夫本莽、用金钱碾压一切。

20. **零号大坝保安 (Dam Guard)**：常年驻扎大坝、对大坝的每个角落了如指掌的“地头蛇”。

21. **长弓狙神 (Longbow Sniper)**：蹲在长弓溪谷山顶一整局不动、千里之外取人首级。

22. **航天基地仓鼠 (Space Rat)**：专挑航天基地犄角旮旯躲着、偷偷塞满背包。

23. **修脚大师 (Toe Clipper)**：喜欢用霰弹枪或冲锋枪专打敌人护甲覆盖不到的腿部。

24. **描边大师 (Outline Master)**：人体描边、一顿操作猛如虎，一看伤害二十五。

25. **医疗兵的爹 (Medic's Daddy)**：疯狂冲锋疯狂倒地，全场都在喊“蜂医救我”。

26. **雷神 (Grenade God)**：包里全是手雷、喜欢用投掷物覆盖战场的爆破狂。

27. **老六 (The Rat)**：极度耐心、永远躲在门后或草丛里阴人的阴险战术家。

## 5. 剧情与题库替换
共设计 30 道全新的战场选择题（选择题库部分），题目情境包括：
- 遭遇突发火力压制时的反应。
- 队友倒地但周围有狙击手时的选择。
- 面对高价值物资（如曼德尔砖）与团队任务冲突时的取舍。
- 攻楼战术选择。
*注意：题目文案设计将更具网感，活泼调皮，融入“坐牢”、“跑刀”、“修脚”、“老六”等玩家常用的三角洲游戏黑话与梗，提升趣味性。*

## 6. 图片资源获取与替换
利用命令行工具或 API 从官方或 AI 绘图工具获取对应新干员及彩蛋的高清立绘图片，保存至 `/workspace/image/delta_ops/` 目录，并在代码中更新 `TYPE_IMAGES` 映射。

## 7. 实施步骤
1. **确认方案**：用户审核本计划与设计方案。
2. **资源准备**：获取补全的 27 名干员/彩蛋的高清图片（使用 AI 生成缺少的梗图）。
3. **样式修改**：保持现有的深色战术主题。
4. **题库扩充/修改**：调整 30 道题目文案，使其更加活泼、调皮，并加入游戏黑话。
5. **逻辑映射**：修改 JS 中的人格字典 `TYPE_LIBRARY`、将 27 个结果重新分配对应的 15 维度 `NORMAL_TYPES` 向量特征及 `TYPE_IMAGES`。
6. **脚本更新**：更新相关的 python 脚本。
7. **测试与交付**：确保功能正常并启动本地服务预览。
